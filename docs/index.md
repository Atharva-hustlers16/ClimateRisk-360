# Welcome to ClimateRisk360

A comprehensive platform for analyzing climate risks and their impact on insurance claims.

## Features

- **Data Ingestion**: Load data from multiple sources including CSV, Excel, and APIs
- **Data Cleaning**: Handle missing values, outliers, and data quality issues
- **Risk Analysis**: Calculate climate risk scores based on various factors
- **Visualization**: Interactive dashboards and reports
- **Machine Learning**: Predictive models for risk assessment

## Quick Start

```bash
# Install the package
pip install -e .

# Run the Streamlit dashboard
streamlit run src/streamlit_app.py
```
